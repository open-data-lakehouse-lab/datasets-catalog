# Weather Datasets

This directory contains the definitions and documentation for weather-related datasets.

## MVP Focus

The first phase of the Open Data Lakehouse Lab focuses on weather open data from Spanish and/or European public sources.

## Candidate Datasets

We are currently evaluating the following candidate datasets for the weather MVP:

* **[AEMET](aemet/)**: AEMET Weather Open Data Candidate.
* **[Meteocat](meteocat/)**: Meteocat Weather Open Data Candidate.
* **[datos.gob.es](datos-gob-es/)**: datos.gob.es Weather Open Data Candidate.
* **[Dades Obertes Generalitat](dades-obertes-generalitat/)**: Dades Obertes Generalitat Weather Open Data Candidate.
* **[data.europa.eu](data-europa-eu/)**: data.europa.eu Weather Open Data Candidate.

For more details on the selection process, see [docs/dataset-selection-process.md](../../docs/dataset-selection-process.md).
